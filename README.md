# Tenons et Mortaises

## Installation

* Installer ProcessWire dans le répertoire de travail.
* Ajouter les plugins (dézippés) Ace Text Editor et MapMarker dans src/site/modules (voir commande suivante ou http://modules.processwire.com/modules/inputfield-ace-editor/ & http://modules.processwire.com/modules/fieldtype-map-marker/ ).
* >> cd src/site/modules && curl -LkSs https://github.com/adamkiss/InputfieldAceEditor/zipball/master -o InputFieldAceEditor.zip && unzip InputFieldAceEditor.zip && rm InputFieldAceEditor.zip && curl -LkSs https://github.com/ryancramerdesign/FieldtypeMapMarker/zipball/master -o FieldtypeMapMarker.zip && unzip FieldtypeMapMarker.zip && rm FieldtypeMapMarker.zip && cd ../../../
* >> npm install
* >> grunt modules // installation des modules
* >> grunt
